    Me he desenvuelto en trabajos en atencion al cliente, mi trabajo actualmente es Encargada de turno en la reconocida heladeria Freddo.     TECHNOLOGIES

    HTML5

    El lenguaje de hipertexto, esencial de todo sitio, aplico la semantica en la sintaxis para un mejor SEO.

    CSS3

    Mi favorito, donde puedo aplicar la creatividad visual y crear estilos, dandole forma a los sitios web.

    SASS

    Preprocesador, de mucha utilidad para reducir y dar mas eficiencia al codigo CSS.

    TAILWIND

    Framework para CSS, se emplea en el HTML y es a base de clases, flexible y adaptable a las necesidades que requiera.

    BOOSTRAP

    El framework mas conocido para CSS, facilita y agiliza el preceso para el maquetado del sitio web.

    GIT

    El sitema de control de versiones mas usado, facilita el manejo de proyectos, y trabajo en equipo.
